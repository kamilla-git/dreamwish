# 🌿 DreamWish: Дневник ваших заветных желаний ✨

![DreamWish Banner](https://images.unsplash.com/photo-1513885535751-8b9238bd3021?w=1200&q=80)

**DreamWish** — это не просто социальный вишлист, это иммерсивное путешествие в мир ваших мечтаний. Мы создали пространство в эстетике зачарованного леса, где каждое ваше желание превращается в магический свиток, а друзья помогают воплотить сказку в реальность.

[Посмотреть проект](https://github.com/kamilla-git/dreamwish) | [Wiki Проекта](https://github.com/kamilla-git/dreamwish/wiki)

---

## 🍃 Магические особенности

*   **🪄 Волшебный поиск (Scraper)**: Просто вставьте ссылку с **Ozon** или **Wildberries**. Наша магия сама подтянет название, актуальную цену и качественное фото товара.
*   **🕯️ Фактор сюрприза**: Владелец списка никогда не узнает, какой подарок зарезервирован или сколько «магической пыльцы» (вкладов) собрано, до самого момента вручения. Сюрприз защищен на уровне кода!
*   **✨ Real-time синхронизация**: Благодаря Socket.IO все изменения (вклады, резервы) отображаются у друзей мгновенно и без перезагрузки страницы.
*   **🧚‍♂️ Совместные мечты**: На дорогие подарки можно скидываться всей компанией. Прогресс-бар наглядно покажет, насколько мечта близка к исполнению.
*   **🌳 Уникальный дизайн**: Кастомная тема «Изумрудный лес» с использованием шрифта **Alice**, золотых градиентов, 3D-фонов с параллаксом и парящих светлячков.

## 🛠️ Технический артефакт (Стек)

### Frontend
*   **Next.js 15 (App Router)** + TypeScript
*   **Tailwind CSS**: Кастомная палитра Emerald/Gold
*   **Framer Motion**: Магические анимации и переходы
*   **Three.js (React Three Fiber)**: Интерактивный 3D фон
*   **Socket.IO Client**: Реалтайм-обновления

### Backend
*   **FastAPI**: Высокопроизводительный асинхронный сервер
*   **SQLAlchemy Async**: Работа с БД (SQLite/PostgreSQL)
*   **Playwright + Stealth**: «Умный» скрейпинг маркетплейсов с обходом защиты ботов
*   **Python-SocketIO**: Серверная часть реалтайма
*   **JWT Auth**: Надежная защита профилей и данных

## 🚀 Запуск магического портала (Локально)

### 1. Подготовка Бэкенда
```bash
cd backend
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --port 8000
```

### 2. Пробуждение Фронтенда
```bash
cd frontend
npm install --legacy-peer-deps
npm run dev
```

## 📜 Продуктовая философия (Product Decisions)

1.  **Защита дарителей**: Если на подарок внесен хотя бы один вклад, владелец **не может его удалить**. Это защищает усилия и средства друзей.
2.  **Анонимность**: Владелец не видит список дарителей и суммы вкладов даже через инструменты разработчика (маскировка данных на уровне API).
3.  **Гибкость**: Если скрейпер не смог найти товар, пользователь всегда может загрузить фото вручную или вставить прямую ссылку.
